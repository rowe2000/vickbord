TicCmd usage
	help   : shows this help
	quit   : quit the app
	abs    : moves to the absolute position to 0
	abs x  : moves to the absolute position to x
	rel x  : moves relative x
	at     : set current position to 0
	at x   : set current position to x
	pos    : print current position
	status : print status
	vars   : print variables
